This package contains the required files for the manual plunge freezer.  This package must be kept intact.  Revisions must be logged in the ‘changes.txt’ file.  

SketchUp files can be opened with the free Sketchup Make (http://www.sketchup.com/products/sketchup-make) application or the free SketchUp Viewer (http://www.sketchup.com/products/sketchup-viewer), available for both Apple OS and Microsoft Windows.  LayOut file can be opened only with the Pro version of SketchUp.
